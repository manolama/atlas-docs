    stack

    Input Stack:
    Query
    		 ⇨ Output Stack:
    StyleExpr

    Change the line style to stack instead of the default of individual lines. In this mode the line will be filled to the previous stacked line on the same axis.

    Example:

    BeforeAfter
    name,sps,:eq,
(,nf.cluster,),:by

    name,sps,:eq,
(,nf.cluster,),:by,
:stack
