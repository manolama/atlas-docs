    srandom

    Input Stack:
    seed: Integer
    		 ⇨ Output Stack:
    TimeSeriesExpr

    Generate a time series that appears to be random noise for the purposes of experimentation and generating sample data. To ensure that the line is deterministic and reproducible it actually is based on a hash of the timestamp. The seed value is used to vary the values for the purposes of creating mulitple different sample lines. Each datapoint is a value between 0.0 and 1.0.

    42,:srandom
