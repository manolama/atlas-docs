    axis

    Input Stack:
    Query
    Integer
    		 ⇨ Output Stack:
    StyleExpr
     

    Specify which Y-axis to use for the line. Axis are indexed with 0 as the left and 1 or greater to the right. If only one time series is present, it will use the left axis regardless of the :axis setting. Likewise, if all time series are assigned to an axis greater than 0, they will be shifted to the left axis.

    Example with one series on the left, another on the right:

    BeforeAfter
    name,requestsPerSecond,:eq,
name,requestLatency,:eq

    name,requestsPerSecond,:eq,
name,requestLatency,:eq,
1,:axis

    Example with three series and three axis:

    Multi Axis
    name,requestsPerSecond,:eq,
1,:axis,
name,requestLatency,:eq,
name,sps,:eq,
2,:axis
