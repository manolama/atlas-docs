    topk-others-max

    Warning

    This operation is still incubating and may change.

    Input Stack:
    TimeSeriesExpr
    stat: String
    k: Int
    		 ⇨ Output Stack:
    TimeSeriesExpr
     
     

    Since: 1.7

    Restrict the output for a grouped expression to the k time series with the largest value for the specified summary statistic and computes a max aggregate for the other time series. Example of usage:

    BeforeAfter
    name,sps,:eq,
(,nf.cluster,),:by

    name,sps,:eq,
(,nf.cluster,),:by,
max,2,:topk-others-max
