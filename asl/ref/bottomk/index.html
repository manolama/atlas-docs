    bottomk

    Input Stack:
    TimeSeriesExpr
    stat: String
    k: Int
    		 ⇨ Output Stack:
    TimeSeriesExpr
     
     

    Restrict the output for a grouped expression to the k time series with the smallest value for the specified summary statistic. Example of usage:

    InputOutput
    name,sps,:eq,
(,nf.cluster,),:by

    name,sps,:eq,
(,nf.cluster,),:by,
max,2,:bottomk

    In some cases it can be useful to see an aggregate summary of the other time series that were not part of the bottom set. This can be accomplished using the :bottomk-others-$(aggr) operators. Permitted aggregations are avg, max, min, and sum. For more details see:

    • :bottomk-others-avg
    • :bottomk-others-max
    • :bottomk-others-min
    • :bottomk-others-sum
